# stackInterop

A Stacks project for cross-chain interoperability experiments.

## Project Structure

- `contracts/`: Clarity smart contracts.
- `tests/`: Vitest-based test suite.
- `settings/`: Deployment settings for different networks.

## Getting Started

1. Install Clarinet: [Clarinet Installation](https://docs.hiro.so/clarinet/how-to-guides/how-to-set-up-local-development-environment)
2. Run tests: `clarinet test`
3. Check contract syntax: `clarinet check`
